# CDMA (Code Division Multiple Access)

**코드 분할 다중 접속(Code Division Multiple Access, CDMA)**는 채널 분할 다중 접속 프로토콜의 일종으로, 각 송신자에게 고유한 **코드(code)**를 할당하여 같은 주파수와 시간을 공유하면서도 동시 통신을 가능하게 한다.

## 동작 원리

### 인코딩

각 데이터 비트 dᵢ는 0 → -1, 1 → +1로 표현된다. 비트 슬롯은 M개의 **미니슬롯**으로 세분되며, 각 미니슬롯에서 데이터 비트에 M-비트 CDMA 코드 cₘ을 곱한다:

```
Zᵢ,ₘ = dᵢ · cₘ    (m = 1, 2, ..., M)
```

코드는 +1 또는 -1 값의 시퀀스이며, 원래 데이터 비트보다 훨씬 빠른 속도(**chipping rate**)로 변한다.

### 디코딩

수신기는 자신의 코드를 사용하여 원래 데이터를 복구한다:

```
dᵢ = (1/M) Σₘ₌₁ᴹ Zᵢ,ₘ · cₘ
```

### 다중 송신자

복수 송신자의 신호는 **가산적(additive)**으로 합쳐진다. 수신기에서 받는 값:

```
Zᵢ,ₘ* = Σₛ₌₁ᴺ Zᵢ,ₘˢ
```

코드가 **직교(orthogonal)**하게 선택되면, 각 수신기는 집합 신호에서 자기 송신자의 데이터만 추출할 수 있다 — 마치 칵테일 파티에서 특정 언어의 대화만 골라 듣는 것과 유사하다.

## 실제 과제

- 코드 직교성 보장이 어렵다
- 여러 송신자의 수신 신호 강도가 동일하다는 가정이 현실에서는 성립하지 않는다 (near-far 문제)
- 무선 LAN과 셀룰러 기술에서 널리 사용된다

## CDMA vs 다른 다중 접속 방식

| 방식 | 분할 대상 | 예시 |
|------|----------|------|
| FDMA | 주파수 | FM 라디오 |
| TDMA | 시간 | GSM |
| **CDMA** | **코드** | **IS-95, WCDMA** |

CDMA는 코드 공간(codespace)을 분할하는 방식으로, 시간이나 주파수 분할과 대비된다.
